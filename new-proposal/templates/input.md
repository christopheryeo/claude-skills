# New Proposal - Required Inputs

## 1. Customer Information
- [ ] Customer/Company Name
- [ ] Industry
- [ ] Primary Contact Name & Title
- [ ] Contact Email
- [ ] Project/Proposal Name

## 2. Proposal Details
- [ ] Proposal Type: Sales Proposal
- [ ] Project Start Date (if known)

## 3. Customer Brief Details
- [ ] Customer Pain Points (3-5 key points)
- [ ] Business Objectives/Goals
- [ ] Technical Requirements
- [ ] Timeline/Deadline

## 4. Solution Components
- [ ] Products/Services to Include
- [ ] Key Features to Highlight
- [ ] Required Customizations
- [ ] Integration Requirements

## 5. Value Proposition
- [ ] Desired Business Outcomes (3-5 points)
- [ ] Success Metrics/KPIs
- [ ] Competitive Differentiators to Emphasize

## 6. Additional Assets (Optional)
- [ ] Case Studies (Industry/Use Case Specific)
- [ ] Technical Diagrams/Architecture
- [ ] Implementation Timeline
- [ ] Team Bios/Resumes

## 7. Delivery & Follow-up
- [ ] Additional Documents to Include
- [ ] Follow-up Meeting Date/Time
- [ ] Next Steps/Decision Timeline

## Notes:
- Required fields must be completed before proposal generation
- Optional fields enhance personalization but are not mandatory
- File naming convention: `<YYYYMMDD>-<CustomerName>-Proposal-R<Revision>`
- Final document will be saved to the appropriate Workday folder
